 LeadLogic_AI
Intelligent Lead scoring & Sentiment Analysis Engine

LeadLogic AI isa product-ready NLP (Natural Language Processing) tool designed to help sales teams proriitize their pipeline. By analyzing the sentiment and tone of incoming customer inquiries, the engine automatically categorizes leads to ensure high-value prospects never slip through the cracks.

Key Features:
. Sentiment Analysis: Uses NLP to to determine the emotional "heat" of a lead message.
. Actionable Insights: Categorizes leads into Hot, At-Risk, or Inquiry status.
. Strategic Recommendations: Suggests immediate business actions(e.g., "Immediate Follow-Up" vs. "Support Escalation").
. Professional Architecture: Features a strict separation of concerns between the core logic and the web interface.

Tech Stacks:
. Python: The core programming language
. Streamlit: For the high-performance web dashboard.
. TextBlob: For processing and scoring natural language data.

Project Structure:
. engine.py: The "Brain" containing the major function and scoring method.
. app.py: The "Face" containing the streamlit UI components and layout.
. requirements.txt: List of dependencies for cloud deployment.

Local Setup:
To run this project locally, clone the repository and install the requirements.
